CSS Spritesheets

Click on Mario to make him jump. Mario is a div set to 512px x 512px with a background image spritesheet that is 512px x 1024px. This is using an onclick method to add/remove the class jump. 

            const mario = document.getElementById("mario");
    
            function ToggleMario() {
                mario.classList.toggle("jump");
            }

            function MakeMarioJump() {
                if (mario.classList.contains("jump")) {
                    ToggleMario();
                }
        
                ToggleMario();
        
                setTimeout(ToggleMario, 500);
            }
        
            .mario {
                height: 512px;
                width: 512px;
                margin: 24px;
                background-image: url(Mario.png);
                background-position: 0 0;
            }
        
            .jump {
                background-position: 0 512px;
            }

You can use setTimeout({your function}, {time in ms}) to cause a delay before a function is called. When you click the button below it calls the MakeMarioJump function and make him toggle to the jump position and after 500ms call the toggle method again to make him return to standing.
